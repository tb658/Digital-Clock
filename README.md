# Digital Clock

## About
A responsive digital clock with city background switching. Real-time 12-hour display with AM/PM and glassmorphism design.

## Demo
Go to https://tb658.github.io/Digital-Clock/ in any browser. Select cities like Tokyo or Paris from the dropdown menu.

## Features
- Live clock updates every 1ms
- 8 city backgrounds (Tokyo, New York, Paris, etc.)
- Responsive centering, backdrop blur effect
- Made with HTML, CSS, and JavaScript
